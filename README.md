# Project Zomboid mods

## Icons inventory

Redesign the original list-based inventory into an icon-based grid.

- Reduces visual overwhelm
- Preserves the vanilla features
- Doesn't add quality-of-life features (not cheating Indie Stone developers' intentions)

## Side-By-Side Containers

Adds options to control containers list position in inventory windows. Puts them side-by-side by default.

## Static Backpacks

Sets a fixed order to equipped containers, making it much easier to find the right place to drop items.
